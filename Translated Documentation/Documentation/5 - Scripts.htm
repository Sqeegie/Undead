SL_engine_ini_begin SL_engine_ini_begin

Commence l'initialisation du système. Starts system initialization. Doit être exécuté une seule fois au début de l'initialisation de l'objet engine (lors par exemple d'un événement Create , Room Start ou Game Start ), et avant tous les autres scripts et variables. Must be executed once at the beginning of the initialization of the engine object (for example during a create event, Room Start Game or Start), and before all other scripts and variables.

SL_engine_ini_end SL_engine_ini_end

Termine l'initialisation du système. Completes system initialization. Doit être exécuté une seule fois à la fin de l'initialisation de l'objet engine , après et dans le même événement que SL_engine_ini_begin . Must be executed once at the end of the initialization of the engine object, and after the same event SL_engine_ini_begin.

SL_engine_render SL_engine_render

Effectue le rendu de la totalité des ombres et lumières. Renders all the lights and shadows. Doit être placé dans l'objet engine dans des événements de type Step (de préférence End Step ), et doit être exécuté avant le script SL_engine_draw . Must be placed in the engine object in Step type events (preferably End Step), and must be performed before SL_engine_draw script.

SL_engine_draw SL_engine_draw

Affiche l'ensemble des ombres et lumières. Displays all lights and shadows. Doit être placé dans l'événement Draw de l'objet engine , et exécuté après le script SL_engine_render . Must be placed in the Draw event of engine object, and executed after the SL_engine_render script.

SL_engine_free SL_engine_free

Libère la mémoire occupée par la totalité des surfaces du système (y compris celles des objets light ). Free memory used by all of the surfaces of the system (including those of light objects). Doit être exécuté lors de la destruction de l'objet engine (dans un événement Destroy , par exemple). To be executed during the destruction of the engine object (in a Destroy event, for example).

SL_light_ini_begin SL_light_ini_begin

Commence l'initialisation d'un objet light . Begins the initialization of an object light. Doit être exécuté une seule fois au début de l'initialisation de cet objet light (lors par exemple d'un événement Create , Room Start ou Game Start ). Must be executed once at the beginning of the initialization of this object light (for example during a create event, or Game Room Start Start).

SL_light_ini_end SL_light_ini_end

Termine l'initialisation d'un objet light . Ends the initialization of an object light. Doit être exécuté une seule fois à la fin de l'initialisation de cet objet light , après et dans le même événement que SL_light_ini_begin . Must be executed once at the end of the initialization of the object light, and after the same event SL_light_ini_begin.

SL_light_free SL_light_free

Libère la mémoire occupée par les surfaces d'un objet light . Free memory by the surfaces of an object light. Doit être exécuté lors de la destruction de ce dernier (dans un événement Destroy , par exemple). To be executed during the destruction of the latter (in a Destroy event, for example).
